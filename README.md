# Bank Management System (Java)

A Java Swing based Bank Management System using:
- Java (Swing, AWT)
- MySQL
- JDBC
- JCalendar

## Features
- User Login & Signup
- Deposit & Withdraw
- Balance Enquiry
- Mini Statement

## How to Run
1. Import project in Eclipse
2. Add mysql-connector-j.jar
3. Add JCalendar-1.4.jar
4. Run Login.java
